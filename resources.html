Here is a collection of resources that can be used to make programming in C++ a much better experience.

CPPReference

CPPReference contains everything in regards to the Standard Template Library (STL) from examples to the code used to write each element of the STL.

CPlusPlus.com

Similar to CPPReferemce, but in a much easier to digest form factor, Cplusplus.com allows for quick reading to get you back into programming.

StackOverflow

The place you go to ask questions when things aren’t working with programming. If there is a problem you are having with your code, chances are someone else has had the same issue and asked it here.

DevDocs

The ultimate collection of anything programming that can be customized to each user’s own preferences. Ranging from C++ to even HTML, this has everything you will need.
