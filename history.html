General

C++ was originally developed in 1985 by Bjarne Stroustrup as an extension of the C programming language. Since there was no official standards written when C++ was released, the only available reference for the language was the book “The C++ Programming Language” written also by Stroustrup.

Version History

C++ has had a variety of versions throughout it’s existance. The most recent public version is C++20 and the testing version is C++23 Nowadays, the version number matches the year it was released but initialy it had versions 1 and 2. C++’s versions are as follows:

  • C++ 1.0 - 1985
  • C++ 2.0 - 1989
  • C++98 - 1998
  • C++03 - 2003
  • C++11 - 2011
  • C++14 - 2014
  • C++17 - 2017
  • C++20 - 2020 (most recent public version)
  • C++23 - 2023 (beta version)
